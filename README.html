Arquitectura del Sistema

El sistema está compuesto por cuatro nodos Python que corren sobre ROS2 Humble en una Jetson Orin, más un nodo C++ de cámara independiente. Cada nodo tiene una responsabilidad única y se comunica exclusivamente mediante tópicos.

turtle_odometry.py

Estima la posición del robot integrando las velocidades angulares de los encoders mediante cinemática diferencial.

sub: /VelocityEncR sub: /VelocityEncL pub: /odom

path_generator.py

Genera la secuencia de puntos objetivo y los publica uno a uno conforme el robot los alcanza por distancia euclidiana.

sub: /odom pub: /next_point

turtle_closeloop.py

Controlador de lazo cerrado de dos fases: primero orienta el robot hacia el objetivo, luego avanza hasta él.

sub: /odom sub: /next_point pub: /cmd_vel_raw

semaforop.py

Detecta el color del semáforo en HSV y modula la velocidad del robot: verde ×1.0, amarillo ×0.5, rojo ×0.0.

sub: /video_source/raw sub: /cmd_vel_raw pub: /cmd_vel

Nodos — Detalle

turtle_odometry.py

Estima la pose del robot a 100 Hz integrando las velocidades angulares de los encoders. Usa QoS de sensor para la conexión con el PuzzleBot real.

ParámetroValorDescripción
r0.052 mRadio de la rueda
L0.18 mDistancia entre ruedas
rate100 HzFrecuencia de integración
v     = r · (ωR + ωL) / 2
ω     = r · (ωR − ωL) / L
x    += dt · v · cos(θ)
y    += dt · v · sin(θ)
θ    += dt · ω  →  normalizado con atan2

path_generator.py

Publica el punto objetivo actual cada 0.5 s. Avanza al siguiente punto cuando la distancia euclidiana al objetivo es menor a 0.1 m.

Trapecio[2,0] → [1.5,1] → [0.5,1] → [0,0]

Instalación y Uso

Crear el paquete

cd ~/ros2_ws/src
ros2 pkg create --build-type ament_python mi_paquete \
  --dependencies rclpy geometry_msgs sensor_msgs turtlesim std_msgs cv_bridge

Copiar los nodos

cp turtle_odometry.py  ~/ros2_ws/src/mi_paquete/mi_paquete/
cp turtle_closeloop.py ~/ros2_ws/src/mi_paquete/mi_paquete/
cp path_generator.py   ~/ros2_ws/src/mi_paquete/mi_paquete/
cp semaforop.py        ~/ros2_ws/src/mi_paquete/mi_paquete/

Registrar entry points en setup.py

entry_points={
    'console_scripts': [
        'turtle_odometry  = mi_paquete.turtle_odometry:main',
        'turtle_closeloop = mi_paquete.turtle_closeloop:main',
        'path_generator   = mi_paquete.path_generator:main',
        'semaforop        = mi_paquete.semaforop:main',
    ],
},

Compilar y lanzar

cd ~/ros2_ws
colcon build --packages-select mi_paquete
source install/setup.bash
ros2 launch mi_paquete launch.xml

Agrega source ~/ros2_ws/install/setup.bash a tu ~/.bashrc para no olvidarlo entre terminales.
